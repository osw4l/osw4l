# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project overview

GitHub profile repo (`osw4l/osw4l`) — a README-driven profile page with terminal aesthetic, dark/light mode, and auto-generated stats. Key files:
- `README.md` — Profile README with terminal aesthetic, dark/light mode support
- `chart-{backend,frontend,cloud}-{dark,light}.svg` — Custom bar charts (6 files)
- `setting.json` — Color config for the 3D contribution graph
- `.github/workflows/profile-3d.yml` — GitHub Action for 3D contrib SVGs
- `profile-3d-contrib/` — Generated 3D contribution SVGs (auto-committed by workflow)

## Repo scan & SVG chart generation

Charts must be generated by scanning ALL repos (public + private) via `gh` CLI, counting real framework/language usage, then producing 6 SVGs following the spec below. Never use estimated data.

```bash
# List all repos
gh repo list osw4l --limit 200 --json name,primaryLanguage

# Detect frameworks per repo (check package.json, requirements.txt, go.mod, mix.exs, etc.)
```

### SVG chart spec

- Width: 420px, bar height: 10px, border-radius: 5px, row height: 44px
- Dark: bg=#0d1117 border=#21262d text=#c9d1d9 title=#00ff41 track=#161b22 sub=#8b949e
- Light: bg=#ffffff border=#d0d7de text=#24292f title=#0f2027 track=#f0f0f0 sub=#57606a
- Font: `ui-monospace, SFMono-Regular, Fira Code, monospace`
- Label column: 115px, max bar width: 250px, percentage after track
- Title format: `> backend --usage` / `> frontend --usage` / `> cloud --usage`
- Bar colors — Backend: Django=#44d7a8 Celery=#37b34a FastAPI=#009688 Go/Gin=#00ADD8 Flask=#888888 Elixir=#8a4baf Rust=#CE422B
- Bar colors — Frontend: Angular=#DD0031 JS/HTML=#F7DF1E(dark)/#b5a000(light) React=#61DAFB(dark)/#0099cc(light) Next.js=#aaa(dark)/#555(light)
- Bar colors — Cloud: Docker=#2496ED AWS=#FF9900 GCP=#4285F4 Kubernetes=#326CE5 Terraform=#7B42BC
